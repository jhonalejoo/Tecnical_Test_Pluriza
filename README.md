# Tecnical_Test_Pluriza

## CriptoApp

App que lista criptomonedas y muestra sus principales características, además convierte la criptomoneda a la moneda local.

## Instalación

1. Descargar el repositorio y dentro de este ejecutar npm install.
2. Ejecutar en Android - react-native run-android.
3. Ejecutar en Ios - cd ios - pod install - react-native run-ios.
